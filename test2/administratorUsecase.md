# 实验2：图书管理系统用例建模
## 1.系统管理用例
### 1.1系统管理用例PlantUML源码如下：
```
@startuml
:系统管理员:-->(登录)
:系统管理员:-->(账号管理)
:账号管理: -->(添加图书)
:账号管理: -->(删除图书)
:账号管理: -->(添加图书借阅者)
:账号管理: -->(删除图书借阅者)
:账号管理: -->(修改图书借阅者)
:账号管理: -->(系统维护)
@enduml
```

**“系统管理”用例流程图源码如下：**
``` 
@startuml
start
:系统管理员;
:登录;
:系统管理;
split
:图书管理;
split
:删除图书;
split again
:添加图书;
end split
split again
:借阅者管理;
split
:图书借阅者修改;
split again
:图书借阅者添加;
split again
:图书借阅者删除;
split again
end split
end split
stop
@enduml
```
